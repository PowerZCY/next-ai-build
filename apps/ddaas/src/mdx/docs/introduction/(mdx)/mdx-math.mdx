### 常用公式·1

<Tabs items={["欧拉公式", "正态分布", "贝叶斯公式", "欧拉-拉格朗日方程", "纳维-斯托克斯方程", "黎曼猜想ζ函数"]}>

  <Tab value="欧拉公式">
    ```math

    ```
  </Tab>

  <Tab value="正态分布">
    ```math
    f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x - \mu)^2}{2\sigma^2}}
    ```
  </Tab>

  <Tab value="贝叶斯公式">    
    ```math
    P(B_j|A)=\frac{P(B_j)P(A|B_j)}{\sum_{i = 1}^{n}P(B_i)P(A|B_i)}
    ```
  </Tab>

  <Tab value="欧拉-拉格朗日方程">     
    ```math
    \frac{\partial F}{\partial y}-\frac{d}{dx}(\frac{\partial F}{\partial y'}) = 0
    ```
  </Tab>

  <Tab value="纳维-斯托克斯方程"> 
    ```math
    \rho(\frac{\partial \mathbf{v}}{\partial t}+\mathbf{v}\cdot\nabla\mathbf{v})=-\nabla p + \mu\nabla^2\mathbf{v}+\mathbf{f}
    ```
  </Tab>

  <Tab value="黎曼猜想ζ函数">     
    ```math
    \zeta(s)=\sum_{n = 1}^{\infty}\frac{1}{n^s}=\prod_{p}\frac{1}{1 - p^{-s}}
    ```
  </Tab>

</Tabs>
