## 高频操作说明
<Callout type="warn"> 建议先试用我的配置文件, 再用快捷键搜索来更改快捷键, 比起搜索命令关键词要好一些 </Callout>

### 工作场景分析

<Tabs groupId="" persist items={["编辑器", "mdx文档"]}>

  <Tab value="编辑器">
    1.  打开编辑器设置,                `⌘ + ,`
    2.  打开快捷键设置,                `⌘ + ;`
    3.  在项目内全局搜索内容,           `⌘ + ⬆ + F`
    4.  在当前文件内搜索内容,           `⌘ + F`
    5.  打开当前文件所在的文件夹,        `⌃ + ⌥ + ⬆ + O`
    6.  文件重命名,                    `⬆ + F6`
    7.  快速定位当前文件的位置,          `⌃ + ⌥ + ⬆ + I`
    8.  展开/折叠项目侧边栏,            `⌃ + ⌥ + ⬆ + ←`
    9.  展开/折叠AI聊天侧边栏,          `⌃ + ⌥ + ⬆ + →`
    10. 展开/折叠命令行终端下边栏,       `⌃ + ⌥ + ⬆ + ↓`
    11. 打开编辑器快捷命令下拉框,        `⌘ + ⬆ + P`
  </Tab>

  <Tab value="mdx文档">
    1. **文字加粗**, 高频使用快捷键           `⌃ + ⌥ + B`
    2. *文字斜体*, 高频使用快捷键             `⌃ + ⌥ + I`
    3. ~~文字删除线~~, 高频使用快捷键         `⌃ + ⌥ + X`
    4. `文字高亮`, 高频使用快捷键             `⌃ + ⌥ + `
    5. 预览md/mdx文档, 高频使用快捷键         `⌃ + ⌥ + ⬆ + E`
    6. 上/下移动整行, 高频使用快捷键           `⌥ + ⬆ + ↑/↓`
    7. 复制当前整行, 高频使用快捷键            `⌘ + D`
    8. 多行批量编辑列选模式, 高频使用快捷键     `⌘ + ⬆ + I`
    9. 复制当前编辑窗口便于Diff               `⌘ + ⌥ + 2`
  </Tab>
</Tabs>
