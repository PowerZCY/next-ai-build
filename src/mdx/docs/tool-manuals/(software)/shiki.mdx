# 支持的语言列表

## Shiki默认图标

### bash

```bash title="shiki.bash"
echo "Hello Re8ger"
```

### c

```c title="shiki.c"
#include <stdio.h>
int main() {{
   printf("Hello Re8ger\n");
   return 0;
}}
```

### c#

```c# title="shiki.cs"
using System;
public class Hello {{
    public static void Main(string[] args) {{
        Console.WriteLine("Hello Re8ger");
    }}
}}
```

### c++

```c++ title="shiki.cpp"
#include <iostream>
int main() {{
    std::cout << "Hello Re8ger" << std::endl;
    return 0;
}}
```

### go

```go title="shiki.go"
package main
import "fmt"
func main() {{
    fmt.Println("Hello Re8ger")
}}
```

### graphql

```graphql title="shiki.graphql"
# Query to greet Re8ger
query GreetRe8ger {{
  greeting(name: "Re8ger")
}}
```

### js

```js title="shiki.js"
console.log("Hello Re8ger");
```

### md

```md title="shiki.md"
# Hello Re8ger

This is a greeting to Re8ger in Markdown.
```


### shell

### ts

