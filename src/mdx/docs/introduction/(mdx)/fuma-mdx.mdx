# Overview
<Callout title="使用方式" type="info">
1. 基于效率的考量, 对于高频使用语法, 一并提供对应的触发关键词, [模板配置](./mdx-snippets)
2. 语法示例会使用3个Tab标签
  - 示例, tab标题为 `example`
  - 源码, tab标题为 `source`, 方便直接copy
  - 触发关键词, tab标题为 `@keywords`
</Callout>
