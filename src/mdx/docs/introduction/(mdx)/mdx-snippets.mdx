## 代码模板

在**VSCode**\(`Cursor`|`Trae`\)中, 使用`cmd+shift+p`打开命令面板, 输入`snippet`即可配置代码模板

<Callout title="VSCode" type="info">
    请务必注意VSCode的版本, VSCode Version: 1.96.2+ 是肯定可以用的, Cursor或Trae都已支持该版本
</Callout>



### 快捷操作
```txt title="mdx-snippets"
# 生成mdx帮助信息
@mdxhelp

# 生成mdx文件头信息
@mdxfm

# 生成mdx提示信息
@mdxmsg

# 生成mdx卡片组
@mdxcg

# 生成mdx单张卡片
@mdxscard

# 生成mdx安装命令Tab
@mdxinstalltab

# 生成mdx标签Tab组
@mdxtabs

# 生成mdx单个标签Tab组子项
@mdxstab

# 生成mdx问答组
@mdxfaq

# 生成mdx问答组子项
@mdxsfaq

# 生成mdx文件系统示意图
@mdxfiles   

# 生成mdx单个文件
@mdxfile

# 生成mdx🔐锁定文件
@mdxlfile   

# 生成mdx文件夹
@mdxfolder

# 生成mdx置灰文件夹
@mdxifolder     

# 生成mdx空文件夹
@mdxefolder

# 生成mdx步骤组
@mdxsteps   

# 生成mdx步骤组子项
@mdxsstep

# 生成mdx代码块
@mdxcode    

# 生成mdx图标Mermaid
@mdxmermaid

# 生成mdx单页面
@mdxspage

# 生成mdx数学公式
@mdxmath
```
