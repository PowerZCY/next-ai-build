## 代码模板

在**VSCode**\(`Cursor`|`Trae`\)中, 使用`cmd+shift+p`打开命令面板, 输入`snippet`即可配置代码模板

<Callout title="VSCode" type="info">
  请务必注意VSCode的版本, VSCode Version: 1.96.2+ 是肯定可以用的, Cursor或Trae都已支持该版本<br/>
  
</Callout>



### 快捷操作
```txt title="mdx-snippets"
@mdxfm           :  生成mdx文件头📝信息
@mdxmsg          :  生成mdx提示💬信息
@mdx1msg         :  生成mdx提示[清爽info]ℹ️信息
@mdxmsgi         :  生成mdx提示[infoℹ️]信息
@mdxmsgw         :  生成mdx提示[warn⚠️]信息
@mdxmsge         :  生成mdx提示[error❌]信息
@mdxmsgs         :  生成mdx提示[success✅]信息
@mdxinstalltab   :  生成mdx安装📦命令Tab
@mdxcg           :  生成mdx卡片组🍻
@mdxscard        :  生成mdx单张卡片🍺
@mdxtabs         :  生成mdx标签📑Tab组
@mdxstab         :  生成mdx单个标签🔖Tab组子项
@mdxitabs        :  生成mdx引入文件📁Tab组
@mdxistab        :  生成mdx单个引入文件📄Tab组子项
@mdxif           :  生成mdx单独引入文件📎
@mdxfaq          :  生成mdx问答组❓
@mdxsfaq         :  生成mdx问答组子项💡
@mdxfiles        :  生成mdx文件系统示意图📁
@mdxfile         :  生成mdx单个文件📄
@mdxlfile        :  生成mdx锁定🔐文件
@mdxfolder       :  生成mdx文件夹📂
@mdxifolder      :  生成mdx置灰文件夹🔒
@mdxefolder      :  生成mdx空文件夹📁
@mdxstep         :  生成mdx步骤①②③
@mdxcode         :  生成mdx代码块💻
@mdxmath         :  生成mdx数学公式π
@mdxmermaid      :  生成mdx图标Mermaid📊
@mdxspage        :  生成mdx单页面📄
```
